# while문

## while문 이해하기

```java
public static void main(String[] args) {
/*
 * 출력예)1
 * 		2
 * 		...
 * 		10
 *
 */
 
  int a = 1;
  while(a<=10) 
  {
    System.out.print(a++);
  }
  System.out.println();

  a = 11;
  do 
  {
    System.out.println(a++);
  }while(a<=10);
}
```

```console
12345678910
11
```
